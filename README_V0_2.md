# Rob v0.2 — base autonómica

Esta versión crea la estructura inicial del metabuscador genealógico.

## Cobertura territorial registrada

Incluye las 17 comunidades autónomas, Ceuta y Melilla.

Importante: que una comunidad esté registrada no significa todavía que su conector esté operativo.
Los estados posibles son:

- `registered`: fuente inventariada, sin conector funcional.
- `development`: conector en construcción.
- `functional`: responde técnicamente.
- `verified`: probado con búsquedas reales y resultados comprobados.
- `disabled`: desactivado por errores o incompatibilidad.
